# Task-01: Population Distribution Visualization

## 📌 Internship
This project is part of the **Prodigy Infotech Data Science Internship**.

## 📊 Objective
The objective of this task is to create visualizations such as **bar charts and histograms** to analyze the distribution of categorical or continuous variables in a dataset.

## 📁 Dataset
The dataset used in this project is the **World Bank Population Dataset**, which contains information about population statistics for different countries and regions.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

## 🔍 Steps Performed
1. Imported necessary Python libraries.
2. Loaded the dataset using **Pandas**.
3. Explored the dataset to understand its structure.
4. Created a **bar chart** to visualize the distribution of countries by region.
5. Analyzed and interpreted the visualizations.

## 📈 Visualizations
The following visualizations were created:
- **Bar Chart** showing the number of countries in different regions.

This visualization help in understanding patterns and distributions in the dataset.
